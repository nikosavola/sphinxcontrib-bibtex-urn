# Changelog

## 0.1.0

*Initial release.*

- URN:NBN identifier support for sphinxcontrib-bibtex.
- Country-specific resolvers for AT, CZ, FI, HR, HU, IT, NL, NO, SE, SI.
- Global fallback via [nbn-resolving.org](https://nbn-resolving.org/).
- Auto-deduplication of redundant `url` fields.
- URL-to-URN promotion for known resolver URLs.
- Pre-built styles: `urn_plain`, `urn_unsrt`, `urn_alpha`, `urn_unsrtalpha`.
- Dynamic style wrapping via the Sphinx extension.
